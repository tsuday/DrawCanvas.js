Class: DrawCanvas

DrawCanvas

new DrawCanvas(canvasEle, undoStackSizeopt)

Canvas users can do painting.
Drawn image can be obtained as Base64 string.
Parameters:
Name Type Attributes Description
canvasEle object canvas element to draw
undoStackSize number <optional>
 Size of stack used for undo function. Negative value indicates that stack size is theoretically inifinite, and default value is -1. When stack size exceeds this value, oldest data in the stack is abandoned. Note that each elements in stack holds copied whole canvas data, so be careful about data size on memory.
